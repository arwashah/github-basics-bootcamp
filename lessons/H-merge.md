# Lesson H: Merge the PR

Merging brings your changes into the default branch.

What it is:
- A merge combines the commits from your branch into the base branch.

Why it is useful:
- Ships completed work to the main line of development.
- Preserves history of how changes were integrated.

Checklist mapping:
- H) Merge the PR

Tip:
- Use the default merge option unless instructed otherwise.
